| T-01.1-T2 | DoS | LLM response could hang or take >90s | mitigate | context.WithTimeout(ctx, 90*time.Second); the framer / dispatcher honor ctx cancellation per Phase 1 ACP-02; Close() runs in deferred order with the timeout context already canceled |

(11) Drain `stream.Chunks` until the channel closes. Count the number of `ChunkKindText` chunks with non-empty `Text.Content`. Use the same select pattern PATTERNS.md shows:
```
var textChunks int
var firstText string
for {
    select {
    case chunk, ok := <-stream.Chunks:
        if !ok {
            goto drained // channel closed
        }
        if chunk.Kind == canonical.ChunkKindText && chunk.Text != nil && chunk.Text.Content != "" {
            textChunks++
            if firstText == "" {
                firstText = chunk.Text.Content
            }
        }
    case <-ctx.Done():
        t.Fatalf("timed out waiting for chunks (ctx: %v)", ctx.Err())
    }
}
drained:
if textChunks == 0 {
    t.Errorf("no ChunkKindText chunks with non-empty content arrived; the agent never responded with text")
} else {
    t.Logf("received %d text chunks; first chunk content (truncated to 80 chars): %.80s", textChunks, firstText)
}
```

(Use a `goto drained` label OR refactor with a `done := false` boolean — pick whichever is cleaner; the Phase 1 SmokeTest's loop style is fine to adapt.)

result, err := stream.Result()

}
