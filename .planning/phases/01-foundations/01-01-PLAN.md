  Must pass shellcheck (pre-commit hook is active).

  REVIEW FIX (Codex MEDIUM — restart race):
  stop() must NOT remove the PID file immediately after kill. Instead, add a bounded wait loop
  (10s max, 100ms polling) using kill -0 $pid to verify the process has exited BEFORE removing
  the PID file and returning. Pattern:
    kill "$pid"
    local i=0
    while kill -0 "$pid" 2>/dev/null && [ $i -lt 100 ]; do
      sleep 0.1
      i=$((i + 1))
    done
    rm -f "$LOOP24_PID"
  This eliminates the race in restart() where start() races with the dying process.
