## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client (untrusted) → canonical.ChatRequest | Inbound request body carries user-supplied text + tool_use.Input maps + tool_result.Content arrays — UNTRUSTED |
| canonical.ChatRequest → PIIRedactionHook.Before | Hook walks the canonical structure; recognizer regex iteration is the trust boundary that decides what counts as PII |
| PIIRedactionHook → pii.Summary (ctx) | Producer-side populate; LoggingHook (slice 3) is the only consumer |
| PIIRedactionHook → modes.go hash mode → external storage (log files) | If hash mode active, hash tags traverse logs — must be HMAC-keyed to resist rainbow tables on log dumps |
