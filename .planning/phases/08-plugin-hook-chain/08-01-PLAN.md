## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| client → adapter | Inbound HTTP carries `X-Request-Id` header (or doesn't); untrusted external input |
| adapter → engine.Run (PreHook chain) | Adapter stamps inbound `X-Request-Id` onto ctx via `WithRequestID` before invoking `engine.Run`; canonical-typed boundary |
| engine → PostHook | After-callback sees assembled or short-circuit response; ctx still carries request_id |
