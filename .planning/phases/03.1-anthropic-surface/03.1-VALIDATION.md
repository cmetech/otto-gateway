## Test Infrastructure

| Property | Value |
|----------|-------|
| **Framework** | `go test` (stdlib `testing` + `httptest` + `go.uber.org/goleak`) |
| **Config file** | `go.mod` (no extra config — stdlib runner) |
| **Quick run command** | `go test ./internal/adapter/anthropic/ -race -count=1` |
| **Full suite command** | `go test ./... -race -count=1` |

| **Estimated runtime** | ~10–20s (quick) ; ~60–90s (full suite with race) |

---

## Sampling Rate

- **After every task commit:** Run the task's `<automated>` verify command (see Per-Task Verification Map below).
- **After every plan wave:** Run `go test -race ./internal/adapter/anthropic/ ./internal/auth/ ./internal/config/ ./internal/server/ ./internal/engine/ -count=1`.
- **Before `/gsd:verify-work`:** Full suite must be green AND `golangci-lint run`, `go vet ./...`, `go-arch-lint check` all pass.
- **Max feedback latency:** ~20 seconds (quick); ~90 seconds (full).
