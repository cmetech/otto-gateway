## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Engine chunk channel → SSE wire | Untrusted text from kiro-cli flows through chunk channel into SSE frames. JSON encoding escapes control characters automatically (RESEARCH.md Pitfall §Mid-stream poisoning). |
| ctx (client connection) → SSE goroutine | Client disconnect must propagate cleanly to the SSE goroutine. No goroutine leak. |
| http.ResponseWriter ↔ goroutine | D-05 — writer touched from EXACTLY one goroutine. No mutex, no race. |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-3.1-SSE | Resource Leak / DoS | `runSSEEmitter` goroutine lifetime | mitigate | D-05 single-goroutine design + `defer ticker.Stop()` + ctx.Done case in select-loop. Verified by `goleak.VerifyTestMain` at package level AND `goleak.VerifyNone(t)` in individual tests including the ctx-cancellation test. |
| T-3.1-RACE | Concurrent access | http.ResponseWriter / sseEmitter state | mitigate | D-05 — writer never escapes the goroutine. blockIndex/currentKind/blockOpen mutated only inside the select-loop. `go test -race` exercises the package and verifies. |
| T-3.1-POISON | Tampering | Chunk text content containing SSE delimiters | mitigate | All payloads go through `json.Marshal` which escapes `\n`, `\r`, quotes, and other control characters. A chunk containing literal `\n\nevent: message_stop\n\ndata: {"type":"message_stop"}\n\n` produces a valid framed SSE event with the malicious literal preserved as JSON-escaped text inside `delta.text`. Tested via property-style input. |
| T-3.1-PARTIAL | Information Disclosure | SSE partial write on client disconnect | accept | When ctx is canceled mid-write, the http.ResponseWriter may have already partial-flushed a frame. | T-3.1-PING | Resource exhaustion | ping ticker firing on dead connection | mitigate | When the client disconnects, the next Flush after the ping triggers a write error; the select-loop's chunk channel close or ctx.Done case fires immediately after. Goleak verifies the goroutine exits. Worst-case: one ping is buffered to a dead socket, no further pings — bounded by the TCP send buffer. PingInterval named const (W7) makes the cadence auditable. |
