# Phase 6.1 Plan 02: Pool Slots Grid + Active Sessions Table Summary

**One-liner:** Pool-slot 3-col CSS grid with dead-slot 2px red border override and active-sessions table with relative-time formatting, both DOM-patched in-place by `renderSlots`/`renderSessions` on every 30s poll.

## What Was Built

### Task 1: Pool Slots 3-column grid section

**internal/admin/templates/index.html.tmpl** — replaced the empty `data-slot-grid` stub with:
- `<div class="otto-slot-grid" data-slot-grid></div>` — JS appends `<article class="otto-slot-card">` children
- `<div class="otto-slot-grid-empty" data-slot-grid-empty hidden>No pool slots configured (set POOL_SIZE)</div>` — shown when `snap.pool.size === 0`

**internal/admin/static/css/admin.css** — appended (no existing rules modified):
- `.otto-slot-card.is-dead` calc padding for stable geometry when 2px border activates
- `.otto-slot-label` — font shorthand per UI-SPEC Subhead ramp (600 16px/1.4)
- `.otto-slot-meta` / `.otto-slot-meta.is-busy` / `.otto-slot-meta.is-dead` — body ramp (400 14px/1.5) with semantic color overrides
- `.otto-badge.is-alive` (bg `#1B3B2E`, color `--otto-healthy`) — pre-computed 12% opacity hex
- `.otto-badge.is-dead` (bg `#3D1E1E`, color `--otto-failed`) — pre-computed 12% opacity hex
- `.otto-session-id-short` — caption ramp + monospace for session IDs
- `.otto-slot-grid-empty` — padding + muted centered text

**internal/admin/static/js/admin.js** — inside existing IIFE:
- `shortId(id)` — truncates ids longer than 10 chars to `XXXX…XXXX`
- `buildSlotLabel(slot)`, `buildSlotBadges(slot)`, `buildSlotMeta(slot)` — DOM node factories using `.textContent` (T-6.1-07 XSS mitigation)
- `buildSlotCard(slot)` / `updateSlotCard(article, slot)` — full build and in-place update
- `renderSlots(slots)` — DOM-patches `[data-slot-grid]`; full rebuild only when `slots.length` changes, otherwise update-in-place; calls `fetchSnapshot` success path after `renderSummary`
- Slot meta copy per UI-SPEC: dead → `Dead — respawning…` (red); busy → `Busy — session {shortId}`; idle → `Idle`
- Badge logic: dead → DEAD red; alive+busy → ALIVE green + BUSY purple; alive+idle → ALIVE green

**internal/admin/handlers_test.go** — added `TestAdmin_PageHandler_PoolGridScaffold`:
- Asserts rendered HTML contains `data-slot-grid`, `data-slot-grid-empty`, `class="otto-slot-grid"`

### Task 2: Active Sessions table section

**internal/admin/templates/index.html.tmpl** — replaced the empty `data-sessions-card` stub with:
- `<div class="otto-sessions-card" data-sessions-card>` containing:
  - `<table class="otto-sessions-table" data-sessions-table>` with thead (Session/Status/Last used/Model) + `<tbody data-sessions-tbody></tbody>`
  - `<div class="otto-empty-state" data-sessions-empty hidden>` with "No active sessions" heading + body copy per UI-SPEC Copywriting Contract

**internal/admin/static/css/admin.css** — appended:
- `.otto-sessions-card` — card surface with `overflow-x: auto` for mobile horizontal scroll
- `.otto-sessions-table` — width 100%, border-collapse: collapse, font shorthand
- `.otto-sessions-table thead th` — caption ramp + muted + uppercase + bottom border
- `.otto-sessions-table tbody td` — md padding + bottom border
- `.otto-sessions-table tbody tr:last-child td` — no bottom border (UI-SPEC last-row contract)
- `.otto-sessions-table td.is-session-id` — monospace font for session IDs
- `.otto-sessions-table td.is-model-null` — muted color for em-dash null model
- `.otto-empty-state` / `.otto-empty-state strong` — centered muted empty state with semibold heading

**internal/admin/static/js/admin.js** — inside existing IIFE:
- `relativeTime(iso)` — pure function: `Xs ago` / `Xm ago` / `Xh ago` / `Xd ago`
- `renderStatusBadge(sess)` — returns `<span class="otto-badge is-dead|is-busy|is-alive">` per session alive/busy state
- `td(cls, text, title)` — `<td>` factory with optional class, text content, and `title=` tooltip
- `buildSessionRow(sess)` — builds `<tr>` with monospace session-id, status badge, relative-time (ISO in title), model/em-dash
- `renderSessions(sessions)` — DOM-patches `[data-sessions-tbody]`; full rebuild when `length` changes; hides/shows table vs empty-state; called from `fetchSnapshot` success path after `renderSlots`
- Session status badge: dead → Dead (red); busy → Busy (purple); idle → Idle (green) per UI-SPEC Copywriting Contract

**internal/admin/handlers_test.go** — added `TestAdmin_PageHandler_SessionsTableScaffold`:
- Asserts `data-sessions-card`, `data-sessions-empty`, `data-sessions-tbody`
- Asserts all four column headers (`<th>Session</th>`, `<th>Status</th>`, `<th>Last used</th>`, `<th>Model</th>`)
- Asserts empty-state copy strings per UI-SPEC Copywriting Contract
