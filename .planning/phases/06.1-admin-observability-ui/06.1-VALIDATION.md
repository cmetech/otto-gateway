## Sampling Rate

- **After every task commit:** Run `go test -race ./internal/admin/...`
- **After every plan wave:** Run `go test -race ./...` (full suite incl. new `internal/server` mount smoke)
- **Before `/gsd-verify-work`:** `make ci` (full lint + race-test + arch-lint + govulncheck) + `make cross` (cross-compile smoke) must be green
- **Max feedback latency:** 30 seconds (admin package quick run)
