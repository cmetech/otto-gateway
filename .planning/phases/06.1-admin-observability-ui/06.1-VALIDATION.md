## Manual-Only Verifications

> Rationale for manual-only classification: Phase 6.1 is a UI phase. The server-side Go code
> (handlers, snapshot aggregator, SSE endpoint, tailer goroutine) is fully covered by automated
> tests under `go test -race ./internal/admin/...`. The manual-only items below all require
> **browser rendering** — they depend on CSS paint results, JavaScript DOM mutations observed
> in a live browser, or real-time streaming behavior with visual inspection. These cannot be
> automated without a headless browser framework (e.g. Playwright/Puppeteer), which is out of
> scope for a no-external-toolchain Go phase.
