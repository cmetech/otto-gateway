## Milestone-level decisions (durable)

- **D-12-01: Two-step Go bump (1.25.0 → 1.26.3 → 1.26.4).** The executor initially targeted 1.26.3 to match the developer's local toolchain. Wave 1's govulncheck run surfaced 2 reachable residuals (GO-2026-5039 in net/http, GO-2026-5037 in x509). Rather than document unreachability, the executor tightened to 1.26.4 — minimum patch level that closes all 23 baseline CVEs. Better outcome: zero residual taints, no `//nolint:gosec` exemptions added. Trade-off: an extra commit + extra `go mod tidy` run. Worth it.
