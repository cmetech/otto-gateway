# Quick 260531-fba: Refactor otto-gw init to render from template

One-liner: Replaced `init_cmd()` heredoc (and PS1 `@"..."@` here-string) with template-copy + `set_env_line` calls so `scripts/.env.otto-gw.example` is the single source of truth for the key list.

## What Was Built

- **`set_env_line FILE KEY VALUE COMMENTED`** (bash): helper that rewrites the `KEY=` line in a file in-place using awk. Handles both commented (`# KEY=val`) and uncommented (`KEY=val`) forms. Anchors on `[^[:space:]]*$` to avoid false-matches on prose comments embedding `KEY=value` in descriptive text.

- **`Set-EnvLine`** (PS1): symmetric PowerShell helper with identical semantics.

- **Template-render flow** (both wrappers):
  1. Verify `$OTTO_SCRIPT_DIR/.env.otto-gw.example` exists — hard-error with the path if not
  2. Strip the instructional header (leading comment block through first blank line)
  3. Prepend a `# Generated by 'otto-gw init'` header
  4. Apply 8 `set_env_line` calls: `AUTH_TOKEN` (commented when auth disabled), `KIRO_CMD`, `HTTP_ADDR`, `ENABLED_HOOKS`, `PII_REDACTION_ENABLED`, `PII_REDACTION_MODE`, `PII_HASH_KEY`, `CHAT_TRACE` (commented when disabled)

- Removed `auth_token_prefix` / `chat_trace_prefix` local variables from bash (and `$authPrefix` / `$chatPrefix` from PS1) — comment state now expressed as the `COMMENTED` argument to `set_env_line` / `Set-EnvLine`.

## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 1 - Bug] Prose comment line matching awk pattern**

- **Found during:** Task 2 implementation (first test run)
- **Issue:** The template contains `# CHAT_TRACE=false (default) means no file is opened on disk...` (line 66). The plan's suggested awk pattern `^[[:space:]]*#*[[:space:]]*KEY=` matched this prose comment, replacing its text with the generated `# CHAT_TRACE=false` value.
- **Fix:** Changed awk pattern from `#*[[:space:]]*key "="` to `#?[[:space:]]*key "=[^[:space:]]*$"` — requires no trailing spaces/words after the value, so prose comments are not matched. Applied symmetrically to the PS1 regex.
- **Files modified:** `scripts/otto-gw`, `scripts/otto-gw.ps1`
